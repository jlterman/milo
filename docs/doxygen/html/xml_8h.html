Detailed Description

This file is the interface to the XML input and output stream The XML output stream is used to serialize the equation tree to an XML file. The XML input stream is to deserialize the XML file into an equation tree

Both streams are designed as state machines. Each class can set expectations on what they see in the input (and throw exceptions) and construct the class object appriopately. Classes also need to construct their XML output correctly or XML state machinge will throw an exception.
